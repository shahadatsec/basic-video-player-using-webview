# Basic Video Player Using WebView

A simple Android practice project built with Java and WebView for learning purposes.

## Features

- Display a list of videos
- Open videos in a WebView
- Multiple Activities
- Click listeners
- Scrollable UI using ScrollView

## Built With

- Java
- XML
- Android Studio
- WebView

## Purpose

This project was created to practice Android development concepts such as:

- XML layouts
- Intents
- Multiple Activities
- WebView integration
- Click listeners
